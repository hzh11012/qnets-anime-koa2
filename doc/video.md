# 视频

## 接口前缀

```shell
http://localhost:5200/api/video
```

## 添加视频 - 管理员

> scope = 3

```
POST    /admin/create
```

### 参数说明

| 参数    | 类型         | 默认值 | 是否必填 | 说明         |
| ------- | ------------ | ------ | -------- | ------------ |
| id      | `number`     | -      | ✅       | 动漫ID       |
| title   | `string(50)` | -      | ✅       | 视频标题     |
| url     | `string`     | -      | ✅       | 视频链接     |
| episode | `number`     | -      | ✅       | 视频集数编号 |

### 成功操作返回

```json
{
    "code": 200,
    "msg": "添加视频成功"
}
```

## 视频播放

> scope >= 1

```

```

### 参数说明

| 参数 | 类型     | 默认值 | 是否必填 | 说明   |
| ---- | -------- | ------ | -------- | ------ |
| id   | `number` | -      | ✅       | 视频id |

### 成功操作返回

```json
{
    "code": 200,
```
